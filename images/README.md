# Bilder für die Entspannungs-Diashow

Bitte fügen Sie die 5 hochgeladenen Bilder in diesen Ordner ein und benennen Sie sie wie folgt:

1. **image1.jpg** - Bergsee mit Kanus (rote Kanus am Steg)
2. **image2.jpg** - Tropischer Strand (Palme, türkisfarbenes Wasser, Liegestühle)
3. **image3.jpg** - Feuerwerk (Feuerwerk über einer Stadt)
4. **image4.jpg** - Regenbogen über Wasser (bunter Regenbogen mit Baum)
5. **image5.jpg** - Welpe in Tasse (süßer Hund in gepunkteter Tasse)

## Format
- Dateiformat: JPG
- Dateinamen müssen exakt wie oben sein (Kleinschreibung beachten!)
- Die Bilder werden automatisch für 10 Sekunden pro Bild angezeigt

## Ablauf
1. Nutzer klickt auf "Weiter" nach Seite 2 ("Wie funktioniert es?")
2. Bildergalerie startet automatisch
3. Jedes der 5 Bilder wird für 10 Sekunden angezeigt (insgesamt 50 Sekunden)
4. Nach dem letzten Bild geht es automatisch zu Seite 3 (Emotionale Zustandsabfrage)
